Try more than once

Try More Than One Start.

A bad first guess can give a bad answer.

K-means begins with random group middles. We score each run by the total squared distance from dots to their group middles. Lower is better, so we keep the run with the lower total.

Make numbers fair

Make Numbers Fair.

Big numbers can drown out small numbers.

K-means uses distance. If one column has a much bigger range, it can dominate that distance. Scaling centers each column and divides by its spread so both count more fairly.

What this method does

Start Alone, Then Join.

Each dot starts alone, then close dots join.

The joins keep going until everything is connected. Later you choose a cut height to turn the tree into final groups.

How groups choose to join

Three Ways Groups Can Join.

Groups can join by the closest pair, average closeness, or farthest pair.

Each rule defines distance between two groups in a different way. Switch the rule and compare the tree on the same dots.

Try this: Switch between closest pair, average closeness, and farthest pair.

Closest, average, or farthest watch how the same dots build different trees
Dots
Tree
Group distance rule average closeness

This is how the method measures distance between two groups before they join. Single uses the closest pair, average uses the mean pair distance, and complete uses the farthest pair. There is no universal higher or lower better choice. Pick the rule that matches your problem.

Practical effect uses the mean distance across all pairs

This tells you what kind of tree the rule often makes. Single can create chains, average smooths extremes, and complete keeps groups tight. Compare the trees and keep the rule that best matches how you want groups to behave.

What does close mean?

What Does Close Mean?

Things can be close by size or close by pattern.

A small rise and a big rise have the same pattern but different sizes. Decide which idea matters first. Later, the technical names are Euclidean distance for size and correlation distance for pattern.

Try this: Switch between Same size and Same pattern.

Try this in R 
d_euclid <- dist(x)
d_corr <- as.dist(1 - cor(t(x)))
hc_corr <- hclust(d_corr, method = "average")
Close by size or by pattern same size groups by amount; same pattern groups by shape
Distance rule same size

This is how the method computes distance between two profiles. Euclidean adds point-by-point size differences. Correlation compares the up-and-down shape after centering each profile. There is no universal higher or lower better choice. Pick the rule that matches your question.

Use this when amount matters more

Use same size when the level matters, like total amount sold. Use same pattern when the rise-and-fall shape matters, even if one line is bigger. Then use the same rule in R.
