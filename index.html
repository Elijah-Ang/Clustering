What is clustering?

Put Similar Dots Together.

Clustering means putting similar things together.

Think of sorting toys into piles that belong together. Here, dots that sit near each other go into the same group.

A Bad First Guess Can Give a Bad Grouping.

A bad first guess can give a bad answer.

K-means begins with a first guess for the group middles. Different first guesses can lead to different final groups, so we try many and keep the best answer.

What this method does

Each Dot Starts Alone.

Each dot starts alone, then close dots join.

The joins keep going until everything is connected. Later you decide where to cut the tree into final groups.

Try this: Look at the full tree before picking any final groups.

Try this in R 
hc <- hclust(dist(scale(df[, vars])), method = "average")
plot(hc)

What does close mean?

What Does Close Mean?

Things can be close by size or close by pattern.

First decide what kind of closeness matters. Then the technical names are Euclidean distance for size and correlation distance for pattern.

Try this: Switch between Same size and Same pattern.

Try this in R 
d_euclid <- dist(x)
d_corr <- as.dist(1 - cor(t(x)))
hc_corr <- hclust(d_corr, method = "average")
